; ->메모
" ->말하기
r ->길 파기
s ->게임 시작
c ->클리어 했으면
f ->실패 했으면
O ->유닛 열기
L ->레벨 열기
x ->읽기 끝

; ;을 쓰면 이 뒤는 읽지 않는다.
; "을 쓴 뒤에는 비서가 전하는 말
; r 길파기 액션
; s 게임 시작
; c 클리어하면 아래를 읽는다. 아니면 l(else)가 나올 때까지 읽어내려간다.
; f 숫자 별을 숫자개 이상 먹었을 때 이벤트. c 아니면 f중 하나만 쓸것.
; O 병사 종류를 연다
; L 레벨을 연다.
; x를 하면 더이상 읽지 않고 패스.
