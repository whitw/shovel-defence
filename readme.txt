MAP 파일에 저장할 것들

1. 지도 생김새
2. 적군시작 위치
3. 웨이브 마다 나오는 적군 배열(최대 5개)
4. 주어진 돈
5. 초기 성의 체력
6. 게임 이벤트;"rscfOLx


몹이 나오는 시간 간격(ms)
	500 250 125
	속1 속2 속3
  
  이제 자동으로 MAP폴더 안에 파일을 넣어줍니다.
  굳이 MAP폴더를 직접 만들 필요가 없어요.
  mkdir만세


  이벤트를 정렬해서 파일에 저장하고
  그 파일에 저장된 대로 읽어오는 방식을 쓰고자 합니다.
  예를들어서

; ;을 쓰면 이 뒤는 읽지 않습니다.
; "을 쓴 뒤에는 비서가 전하는 말
; r 길파기 액선
; s 게임 시작
; c 클리어하면 아래를 읽는다. 아니면 l(else)가 나올 때까지 읽어내려간다.
; f 숫자 별을 숫자개 이상 먹었을 때 이벤트. c 아니면 f중 하나만 쓸것.
; O 병사 종류를 연다
; L 레벨을 연다.
; x를 하면 더이상 읽지 않고 패스.
"안녕하세요! 새로 오신 걸 환영합니다!
"저는 앞으로 성을 지키는 걸 도와드릴 비서입니다!
"우선 시작해볼까요, 우리 마을은 무역을 주로 하는 도시라
"길을 이어주어야 먹고 살 수 있답니다.
"붉은 마름모에서 보라색 문으로 길을 이어봅시다!
"이 길을 통해 적도 들어올 테니, 길면 길 수록 좋아요.
r
;길 파기를 시작
"좋아요, 이제 잠시 후에 적들이 몰려올 거예요. 화이팅!
s
;게임을 시작
c
;클리어 했을 시에 할 이벤트
O 2
;2번 병사를 연다. 여기서는 궁수.
L 레벨2
;파일 '레벨2'를 level.txt에 추가한다. 
x
;이래서 ifclear또는 if가 아니면 else까지 가야 한다.
l
;클리어 하지 못했을 때 할 이벤트
x
