1. 지도 생김새
2. 적군시작 위치

4. 주어진 돈

6. 게임 이벤트;"rscfOLx

몹이 나오는 시간 간격(ms)
	500 250 125
	속1 속2 속3  

  이벤트를 정렬해서 파일에 저장하고
이를 한줄씩 읽어가면서 게임을 진행한다. 예를 들면 이런것.

; ;을 쓰면 이 뒤는 읽지 않는다.
; "을 쓴 뒤에는 비서가 전하는 말
; r 길파기 액션
; s 게임 시작
; c 클리어하면 아래를 읽는다. 아니면 l(else)가 나올 때까지 읽어내려간다.
; f 숫자 별을 숫자개 이상 먹었을 때 이벤트. c 아니면 f중 하나만 쓸것.
; O 병사 종류를 연다
; L 레벨을 연다.
; x를 하면 더이상 읽지 않고 패스.
